## 6. 반응형 동작 원칙

### 6.1 레이아웃 변화
- 데스크톱: 2컬럼 레이아웃, 사이드바 고정
- 태블릿: 유동적 1-2컬럼 레이아웃, 필터는 수평 스크롤
- 모바일: 단일 컬럼 레이아웃, 중요하지 않은 요소는 접이식 메뉴로 변환

### 6.2 내비게이션
- 데스크톱: 전체 메뉴 표시
- 태블릿: 주요 메뉴만 표시, 나머지는 드롭다운
- 모바일: 햄버거 메뉴로 모든 내비게이션 통합

### 6.3 컨텐츠 조정
- 이미지: 반응형 크기 조정
- 글꼴: 화면 크기에 따라 크기 조정
- 여백: 뷰포트 크기에 따라 자동 조정

### 6.4 상호작용
- 터치 영역: 모바일에서 최소 44x44px
- 호버 효과: 터치 디바이스에서는 탭 상호작용으로 대체
- 스크롤: 모바일에서 자연스러운 스크롤 동작 구현
