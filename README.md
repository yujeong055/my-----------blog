# My Blog

개인 블로그 프로젝트입니다. Next.js와 TypeScript를 사용하여 개발되었습니다.

## 주요 기능

- 마크다운 기반의 블로그 포스팅
- 댓글 시스템
- 사용자 인증 (Clerk)
- 데이터베이스 (Supabase)
- 반응형 디자인

## 기술 스택

- Next.js 14
- TypeScript
- Tailwind CSS
- Clerk (인증)
- Supabase (데이터베이스)
- React Markdown

First, run the development server:

